6.3 传导逻辑：为什么这些事对紫金重要（拆到底）

  • 链条 A：美伊协议 → 油价崩 → 通胀预期降温 → 金价。霍尔木兹要重开 → 此前停产的原油供应预期恢复 → 油价从 92 跌到 83 → 美国通胀最大推手（能源）回落 → 市场预期美联储不必再加息 → 加息预期降温 = 持有黄金的机会成本下降。黄金本身不生息，拿着它就放弃了存钱吃利息的收益；当大家预期利率不再升时，"放弃的利息"变少了、拿金的代价就低了，于是更多人买金 → 金价涨。这就是"地缘缓和但金价反涨"的真正原因。
  • 链条 B：弱美元 → 金/铜双双受益。金和铜都以美元计价，美元贬值时、同样多的实物需要更多美元去买，美元标价的金价铜价被动抬升；弱美元也利好非美买家（如中国铜需求方）。
  • 链条 C：金价 → 紫金利润 → 股价。金价从 4100 涨到 4300+，每盎司多卖约 200 美元，而开采的全维持成本基本固定，多出来的卖价几乎全变利润——这就是"经营杠杆"，收入涨一点、利润涨更多。所以黄金板块能领涨。同花顺·经营杠杆
  • 最大风险点（向下）：以上利多全建立在"加息预期降温"的押注上，6/18 凌晨点阵图才是真裁判。若沃什首秀偏鹰（删除降息、暗示加息），实际利率上行 → 持金机会成本回升 → 金价可能急跌（此前曾有单日跌 188 美元的剧烈波动），紫金黄金板块利润弹性反向收缩。同花顺·风险提示
